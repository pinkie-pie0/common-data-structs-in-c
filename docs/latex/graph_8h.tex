Forward declaration for the graph data structure. Internally implemented as an weighted undirected graph that uses a nested hashmap as the adjacency list (think unordered\+\_\+map$<$node, unordered\+\_\+map$<$node, double$>$$>$, such syntax doesn\textquotesingle{}t exist in C but that\textquotesingle{}s how the graph is implemented, almost). There can only exist at most one symmetric edge between any two vertices.
