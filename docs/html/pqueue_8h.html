typedef struct pqueue_ds pqueue_ds

Forward declaration of the pqueue data structure. Internally implemented as a 4-ary heap with a hashmap ensuring no duplicate elements (think unordered_map<element, index>). Elements are enqueued accordingly to the given comparator function.
