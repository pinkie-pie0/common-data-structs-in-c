◆ avltree_ds

typedef struct avltree_ds avltree_ds

Forward declaration of the avltree data structure. Internally implemented with a nested struct pointer (node containg the value and its left and right children). In addition, the node contains a height variable as it helps the tree keep balance whilist insertions and deletions occur.

◆ avltree_ds_iterator

typedef struct avltree_ds_iterator avltree_ds_iterator

Forward declaration of an avltree iterator. Internally implemented with a deque that iterates through the nodes in an in-order traversal.
