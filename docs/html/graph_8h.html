◆ graph_ds

typedef struct graph_ds graph_ds

Forward declaration for the graph data structure. Internally implemented as an weighted undirected graph that uses a nested hashmap as the adjacency list (think unordered_map<node, unordered_map<node, double>>, such syntax doesn't exist in C but that's how the graph is implemented, almost). There can only exist at most one symmetric edge between any two vertices.
